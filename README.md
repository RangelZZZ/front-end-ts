## Install brew

If your machine already installed yarn, please skip this step.

```shell
brew install yarn
```

## Install dependency packages

```shell
yarn
```

## Running

Run tslint

```shell
yarn tslint
```

Run tslint --fix

```shell
yarn tslint-fix
```

Start project

```shell
yarn start
```

Run test

```shell
yarn test
```
